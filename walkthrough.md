## Design Decisions

| Element | Choice | Rationale |
|---|---|---|
| Name | PROVENANCE | Object history — known to connoisseurs |
| Font | Cormorant Garamond (italic) + Jost 300 | Closest to Delvaux/Valextra editorial tone |
| Color | Warm ivory · deep slate · burnished gold | No cool grays, no neons — Moynat warmth |
| Grid borders | 1px hairlines only | Valextra — form is structure |
| Icons | Roman numerals I. II. III. IV. | No emoji, pure typographic |
| Product cards | CSS color blocks with italic initials | Museum vitrine, not mall shelf |
| CTA copy | "Enquire" not "Buy Now" | The price is never rushed |
| Footer close | *"For those who know."* | IYKYK without saying IYKYK |
